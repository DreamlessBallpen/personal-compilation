# personal-compilation
Compilation of text and python files for workflows and learnings.
The idea is to centralize all of these tidbits squirreled away into their own folders.
Have a central place to browse through, something that is useful for me personally.
There's something magical about being able to do things by following instructions.
There's also something magical about modifying the instructions you've set for yourself,
after trying them out and encountering things that validate, contradict,
or give nuances to what you know.
